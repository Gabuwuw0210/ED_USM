Instrucciones de compilación : 

-Para compilar el primer programa :
g++ Naipe.cpp main.cpp -o output
./output

-Para el segundo : 
g++ Juego.cpp main2.cpp -o programa
./programa

Forma de compilacion : 

Ambos comandos de compilación que proporcionaste utilizan g++, que es el compilador de C++ en la mayoría de los sistemas basados en Unix. Los archivos fuente se compilan en ejecutables especificados después de la opción -o.
